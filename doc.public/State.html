Module State

module State: sig .. end

Representation of player state. This module represents the player state of the card game as it is being played, including the player's hp, deck, hand, and functions that cause the state to change.

type t

The abstract type of values representing a player state.

val get_name : t -> string

get_name t is the name of player in t

val get_hp : t -> int

get_hp t is the hp of player in t

val get_level : t -> int

get_level t is the level of player in t

val get_dazed : t -> int

get_dazed t is the number of turns player is dazed in t

val get_hand : t -> Hogwarts.spell_info list

get_hand t is the hand of a state t

val get_deck : t -> Hogwarts.spell_info list

get_deck t is the deck of a state t

val get_prolong_damage : t -> int list

get_prolong_damage t is the list of prolong damage that will be applied

val get_prolong_turn : t -> int list

get_prolong_turn t is the list of turns of prolonged damage

val get_blocked : t -> int

get_blocked t is the representation of how many times a player is blocked

val get_level_deck : t -> t

get_level_deck t is the updated deck of a state t, filtered out by removing cards that have greater level than the level of player in t

init_player Hogwarts.t name in the initial state of t of player

val init_player_with_level_deck : Hogwarts.t -> string -> t

init_player_with_level_deck Hogwarts.t name is a player initialized with a leveled deck.

val init_enemy : Hogwarts.t -> string -> t

init_enemy Hogwarts.t name in the initial state of t of enemy

val init_enemy_with_level_deck : Hogwarts.t -> string -> t

init_enemy_with_level_deck Hogwarts.t name is an enemy initialized with a leveled deck.

draw t updates the deck of a state t after draw

val update_damage : t -> Hogwarts.spell_info -> t

update_damage t updates the hp of a state t after cast

val update_caster : t -> Hogwarts.spell_info -> t

update_prolong Hogwarts.spell_info returns the list of prolonged damaged after one turn

update spell st1 st2 updates a state t of player and a state t of enemy according to the casted spell
