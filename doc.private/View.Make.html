Module View.Make

module Make: sig .. end

Make creates a view structure for usage.

val print_clear : int -> unit

print_clear len prints out len number of newlines on screen.

val get_house : string -> ANSITerminal.style

get_house house_name gets the respective color code for the house of house_name. Defaults to white if house is not found

val inverse_colour : string -> ANSITerminal.style list

inverse_colour house takes in a house and returns the inverse of the usual. If not one of four house colours (red, green, blue, yellow) then default ANSITerminal style is returned.

val even_odd : ANSITerminal.style list ->
       ANSITerminal.style list -> int -> ANSITerminal.style list

even_odd even odd num returns a list of styles based on if the num is even or odd

val get_color_code : int -> ANSITerminal.style list

get_color_code num gets the respective color of house in accordance with num. 0 or 1 is red, 2,3 yellow, 4,5 green, 6,7 blue. Any other number will return white.

val ref_num : int Pervasives.ref

ref_num counts the current iteration for color coding the title.

val print_arr : int -> (int -> ANSITerminal.style list) -> string list -> string list

print_arr arr prints he first element of a non-empty list and returns all non-printed elements (can be an empty list).

val print_arr_2d : int -> (int -> ANSITerminal.style list) -> string list list -> unit

print_arr_2d modd fn arr prints all elements from a list of string lists. The printing pattern is it prints the head element of each respective list and iterates forward until all lists are empty.

val print_formatted_lst : string -> string list -> unit

Prints formatted list. For more details go to View.View.print_formatted_lst.

val chop : int -> string list -> string list

chop width lst chops a list's strings into one string until no more can be done, then returns the same list with the new string at the head.

val partition_list : int -> string list -> string list

partition_list width lst partitions a list in such a way that all strings in returned list are less than or equal to width characters.

val get_lines : ?pattern:string -> int -> string

get_lines width generates width amount of dashes for ascii card.

val get_centered_text : int -> string -> string

get_centered_text width input returns a string of set width with input at its center.

val pad : int -> string -> string

pad width s returns a string that is padded to be exactly width size.
